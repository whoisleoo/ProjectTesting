## Sobre o Projeto

**Campo Guia** é um projeto originado de uma ideia de um amigo meu chamado <a href="https://github.com/niyjn">KSGabe</a>, o projeto se originou primeiramente com uma tentativa de busca por BOT de discord, você pode estar encontrando o repositório antigo <a href="https://github.com/whoisleoo/Discord-Bot">aqui</a>. Mêses depois surge **Campo Guia**, uma tentativa aprimorada utilizando Web-Scrapping com envio de diversos tipos.

## Instalação e Execução

### ➤ Inicialização do Front-end

```bash
# Clone o repositório
git clone https://github.com/whoisleoo/Campo-Guia.git

# Entre no diretório frontend
cd frontend

# Instale as dependências
npm install

# Inicie o projeto
npm run dev

# Acesse o website
https://localhost:5173

````

### ➤ Inicialização do Back-end
```bash

# Entre no diretório backend
cd backend

# Instale as dependências
npm install

# Inicie o projeto
node server.js

-------------------------------------------

Pra envio via email é necessário configurar o arquivo .env

GMAIL_EMAIL="Email da sua conta"
GMAIL_PASSWORD="Senha de aplicativo do seu email"


````
